Fun Workshop 是一个分享计划，最重要的就是 For fun！
我们希望能网罗到各类有趣的声音来做分享，为我们的学习、研究、工作增添一些些明亮～

Fun Workshop 旨在提供诚恳的信息与见解，因此我们计划为分享者表示充分的尊重。我们期待分享者在活动前知会其将使用的材料，但我们不因除法律自觉外的其他因素限制分享者的自决。
目前，已征集到的分享话题囊括多个领域，AI、商业、交互设计、个人成长、非主流视角等。我们珍视各位分享者的善意与勇气。
因此，为了维护社区氛围，我们谨建议各位不传播社区内讨论信息，少评判分享者表达。目前我们仅面向西交利物浦大学校园进行活动宣传。

Fun Workshop 支持中英双语环境。在原生支持语言翻译的环境（如，微信），我们发布的信息采用使用人数多的语言；在非原生支持语言翻译的环境，我们提供双语材料（如，海报），或实时语言翻译服务（如，在活动现场利用腾讯会议内置翻译功能）。
对于线上会议，需要说明的是，Fun Workshop 并不保证所有活动都提供远程线上参与活动，这是出于包括话题敏感度在内的多种考量。

Fun Workshop 有计划在争得同意后采集、存储并编辑活动现场分享者的音视频数据，并经过明确确认后用于适度的传播分享。但我们不在未进行明确告知并获得明确授权前采集任何其他数据。
Fun Workshop 自愿采用西交利物浦大学学术伦理标准，公示相应文件，并欢迎各位监督。

Fun Workshop 已经搭建了网站用于长期信息共享。各类信息均将同步在微信群与网站发布。网站地址：https://billshiyaozhang.github.io/fun-workshop/

目前 Fun Workshop 仍是志愿运行。在我们鼓励各位加入运行团队的同时，也提醒慎重。
